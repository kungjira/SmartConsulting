Sage 300 Recognized Training Center

บริการฝึกอบรมระบบ Sage 300 ERP ทุก Module และฝึกอบรมการใช้งานของระบบ Crystal Report Professional ทุกวันจันทร์ พุธ และ ศุกร์ ตามตารางของบริษัทฯ

บริษัท สมาร์ท คอนซัลติ้ง จำกัด เปิดให้บริการฝึกอบรมการใช้ระบบ Sage 300 ERP ด้วยห้องอบรมที่ทันสมัยและอุปกรณ์ครบครัน สามารถรองรับผู้เข้าอบรมได้จำนวนมาก รวมถึงท่านสามารถลงทะเบียนเข้าร่วมฝึกอบรมออนไลน์ โดยจะฝึกอบรมผ่าน Web Meeting ทางบริษัทมีทีมงานผู้สอนที่ชำนาญและเชี่ยวชาญในระบบ Sage 300 ERP ทุก Modules ซึ่งได้รับการยอมรับจาก Sage Singapore PTE. LTD. โดยผ่านการทดสอบจนได้รับ Sage 300 Certified Application Consultants ถึง 5 คน

ท่านสามารถส่งพนักงานหรือเจ้าหน้าที่ของบริษัทท่านเข้าฝึกอบรมได้ที่สำนักงานอาคารของบริษัท สมาร์ท คอนซัลติ้ง จำกัดหรือทางอีเมล sales@smartconsulting.co.th ได้ตลอดเวลา 1 ปีแรกที่ซื้อ Sage 300 ERP Application Software จากเรา หรือตลอดสัญญา การให้บริการ Technical Support Services ตามตารางการอบรมของบริษัท ซึ่งท่านสามารถสำรองที่นั่งล่วงหน้าได้
